What is CDN ?

CDN stands for Content Delivery Network. It's a group of servers that are connected
to each other and are spread out across the world. CDNs help to speed up the delivery of web content
to users by storing copies of the content on servers that are closer to the user

crossorigin attribute ?

The crossorigin attribute sets the mode of the request to an HTTP CORS Request.
CORS stands for Cross-Origin Resource Sharing, and is a mechanism that allows resources on a web page 
to be requested from another domain outside their own domain. 
It defines a way of how a browser and server can interact to determine whether it is safe to allow 
the cross-origin request.
